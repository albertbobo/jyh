项目二 车辆清洗操作技能考核评分表

项目名称 车辆清洗 姓名   考核
时限 		45min
细则号 考核要求 配分 等级 评  分  细  则
2.1
汽车外部清洗操作 		使用移动式清洗机对一辆轿车的车体、风档等处以正确的水温、水压；按正确的程序进行清洗（不含车顶部分）。完成后汽车外部应无尘土、无污渍、无水痕，玻璃光亮如新。 20 A 工艺、程序准确，清洗质量优良
B 工艺、程序有部分错误，清洗质量优良
C 工艺、程序有部分错误，清洗质量一般
D

工艺、程序有较多错误，清洗质量一般及以下

备注 1．清洗程序：清洗时应遵循自上而下、先首后尾的操作原则。采用顺时针清洗（也可逆时针清洗）时的操作程序如下：
车顶→前风挡→机盖→左翼子板→中网及车灯→前保险杠→右翼子板→右前轮→右前门→右后门→右后轮→右后部→后风挡→后机盖→后边（保险杠、牌照及车灯）→左后部→左后门→左后轮→左前门→左前轮。
2．清洗水压：第一遍冲洗时应采用雾状水流或扇形喷嘴，水枪的清洗压力应控制在0.2～0.4MPa，冲洗汽车下部的轮胎、底盘等处时，可采用柱状水，压力可再高些。如调压不便可观察学员是否通过调节喷枪与目的物之间的距离来实现对水压的控制。
3．擦干操作：擦干顺序与清洗程序相似，但车顶擦干不作考核。
4．擦干手法：擦干手法为：用洗净拧干的小毛巾对折平铺裹紧整个右手手掌，再用左手抓紧毛巾四边同时将左手压在右手手背上进行擦干。在擦干时应先擦最能够得着范围的四个边，收边后手掌再以横向运动的方式进行擦干。此时要一掌接一掌地横擦，横擦时二脚要呈八字弓步站稳于车旁，同时，腰身要正，视线要紧盯所擦之处，手掌要有力。
5．毛巾分开：擦洗装饰线以下部分将毛巾分开使用—即前后左右为一块毛巾，四个轮毂为一块毛巾。
6．操作要点：高压冲洗前，须检查车窗，前后盖板是否关闭良好。在冲洗车辆时，不要将水冲进钥匙孔。清洗发动机室时，注意不要将水溅到分电器、点火线圈等电气系统的零件上。
7．清洗质量：漆面及外部饰件应无尘土、无污渍、无水痕，玻璃光亮如新，无划痕。
2.2
汽车内室篷、壁清洗操作 		对一辆轿车的内室篷或壁处随机选择一块面积约为500mm×500mm进行清洗。完成后被清洗部分应无污渍遗留痕迹并无因清洗而造成的损伤。 20 A 工艺、程序准确，清洗质量优良
B 工艺、程序有部分错误，清洗质量优良
C 工艺、程序有部分错误，清洗质量一般
D 工艺、程序有较多错误，清洗质量一般及以下
备注 1．用吸尘器吸除篷壁上的浮灰。
2．用干净湿布擦拭，遇有较重污渍，可使用专用的清洁剂予以清洁。
3．清洗污渍时，应从污渍边缘逐渐向中心擦拭。
4．污渍清除干净后，用另一块干净的棉布顺着绒毛方向抹平，使其恢复原样。
5．清洗时抹布不能太湿。
2.3汽车仪表台清洗操作 对一辆轿车的仪表台进行清洗。完成后被清洗部分应无污渍遗留痕迹并无因清洗而造成的损伤。 20 A 工艺、程序准确，清洗质量优良
B 工艺、程序有部分错误，清洗质量优良
C 工艺、程序有部分错误，清洗质量一般
D 工艺、程序有较多错误，清洗质量一般及以下
备注 1．拔下点火开关的钥匙。
2．用吸尘器除去置物箱、空调风道口及汽车收音机周边的污渍，并擦拭干净。
3．用干净软布及棉签（主要用作清除细小条纹处）清除仪表台面、表盘、方向盘、音响、排档区等处的污渍。
4．将专用仪表台清洁剂喷于软布，并轻轻擦拭仪表台各处。
2.4
汽车座椅清洗操作 		对一辆轿车的座椅进行清洗。完成后被清洗部分应无污渍遗留痕迹并无因清洗而造成的损伤。 20 A 工艺、程序准确，清洗质量优良
B 工艺、程序有部分错误，清洗质量优良
C 工艺、程序有部分错误，清洗质量一般
D 工艺、程序有较多错误，清洗质量一般及以下
备注 1．用长毛刷子和吸尘器配合，一面刷座椅表面一边用吸尘器的吸口把里面的灰尘及砂粒清除干净。
2．用吸尘器清洁座垫上的灰尘。
3．将皮革表面用软布擦试干净。
4．将皮革清洗剂均匀地喷敷于皮革表面。稍停片刻，用干净毛巾或软刷子仔细擦试，直至污渍全部清除。
5．应注意学员对椅缝处灰尘及污渍的清除。
2.5汽车安全带清洗操作 对一辆轿车的安全带进行清洗。完成后被清洗部分应无污渍遗留痕迹并无因清洗而造成的损伤。 20 A 工艺、程序准确，清洗质量优良
B 工艺、程序有部分错误，清洗质量优良
C 工艺、程序有部分错误，清洗质量一般
D 工艺、程序有较多错误，清洗质量一般及以下
备注 1．用牙刷蘸淡肥皂水对安全带进行擦洗。
2．清洗后安全带及搭扣处应无任何污渍残留。
3．安全带清洗完毕后不能立即进行卷带（以防发生霉变）。

注：考核项目评分的等级比值如下：A=项目配分×1.0；B=项目配分×0.8；C=项目配分×0.6；D=项目配分×0。
